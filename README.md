# Blitz Inteligente de Telemetria

Projeto desenvolvido para automatizar a auditoria de telemetrias de veículos utilizando Python, Streamlit e Inteligência Artificial.

## Objetivo
Validar divergências entre telemetria nativa e Retrac, automatizando a blitz mensal realizada manualmente pela operação.

## Tecnologias
- Python
- Streamlit
- Pandas
